Data Processing Project

Welcome to the Data Processing Application

This project demonstrates an automated data processing workflow. It involves transforming an Excel file (`data.xlsx`) into a CSV format (`data.csv`), processing it with a Python script (`execute.py`), and publishing the results (`result.json`) via GitHub Pages.

The entire process is orchestrated by a GitHub Actions CI/CD pipeline, ensuring continuous integration and delivery of the processed data.

Key Components:

  • data.xlsx: Original Excel data source (conceptual).
  • data.csv: Converted CSV data (committed).
  • execute.py: Python script for data processing (committed).
  • result.json: Output of the processing script (generated in CI).
  • GitHub Actions: CI/CD pipeline for automation (committed).

How it Works:

  1. `data.xlsx` is conceptually converted; `data.csv` is provided.
  2. `execute.py` reads `data.csv`, performs operations, and outputs `result.json`.
  3. GitHub Actions ensures code quality (ruff) and automates these steps.
  4. `result.json` is published to GitHub Pages.
